Как это работает?

Это программное обеспечение написано на С++ с использованием WinAPI. давайте Розберемся как оно работает!
Cначала включил необходимые модули для работы программы: 

 
#include <windows.h> 
#include <tlhelp32.h>

Работает это таким образом, что мы читаем и списываем в память данные с помощью функции ReadProcessMemory(...) для чтения и WriteProcessMemory(...) для записи.

ReadProcessMemory: 

 
BOOL ReadProcessMemory( 
  HANDLE  hProcess,
  LPCVOID lpBaseAddress,
  LPVOID  lpBuffer,
  SIZE_T  nSize,
  SIZE_T  *lpNumberOfBytesRead
);

WriteProcessMemory: 

 
BOOL WriteProcessMemory( 
    HANDLE hProcess,
    LPVOID lpBaseAddress,
    LPCVOID lpBuffer,
    SIZE_T nSize,
    SIZE_T* lpNumberOfBytesWritten
  );

Поочередно найдем нужные аргументы. Так, для начала у нас HANDLE hProcess:


  HANDLE hProcess ; 
  hProcess = OpenProcess(PROCESS_ALL_ACCESS, 0, ProcessId); // ReadProcessMemory
  hProcess = OpenProcess((0x0008) | (0x0020) | (0x0400), 0, ProcessId); // WriteProcessMemory

Дальее LPVOID lpBaseAddress - это идентификатор процесса(Process id). его можно найти используя FindWindowW или написать функцию самом:


  DWORD FindProcessId (LPCTSTR ProcessName)            
  {
      PROCESSENTRY32 pt;
      HANDLE hsnap = CreateToolhelp32Snapshot (TH32CS_SNAPPROCESS, 0);
      pt.dwSize = sizeof (PROCESSENTRY32);
      if  ( Process32First (hsnap, &pt)) { 
        do  {
            if  (!lstrcmpi (pt.szExeFile, ProcessName)) {
                CloseHandle (hsnap);
                return  pt.th32ProcessID;
              }
          } while  (Process32Next(hsnap, &pt));
      }
      
      std::cout << "Process not found (return 0)" << std::endl;
      CloseHandle (hsnap); 
      return 0;
  }

Мы дошли до адресу LPVOID lpBaseAddress. Это объект, указывающий на определенные данные, но не хранящий их. Например:


DWORD Address = 0x22fe00; // (LPVOID)Address

Осталось только указать буфер и его размер. Буфер должен иметь тип как и у элемента, который вы читаете из памяти; размер буфера указывает с помощью sizeof(buffer)
